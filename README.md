# Mobile Appointment Booking UI – Activ Roots Fitness Studio

## Project Description
This project is a UI/UX case study and mobile website design for an appointment booking system created for Activ Roots – Foundation First Fitness Studio located in Hyderabad.

The goal of this project is to design a simple, modern and user-friendly mobile interface that allows users to explore fitness programs and book training sessions easily.

## Problem Statement
Many local fitness studios still depend on manual booking methods such as phone calls, WhatsApp messages or physical registers.  
This can result in missed bookings, scheduling confusion and poor user experience.

## Project Objective
The main objective of this project is to design a mobile-first booking interface that:

- Helps users explore available fitness programs  
- Allows quick and easy session booking  
- Provides booking confirmation and session tracking  
- Improves overall customer experience  

## Target Users
- Adults interested in strength and fitness training  
- Children enrolling in gymnastics programs  
- Individuals focusing on yoga, pilates and mobility  
- Beginners looking for structured fitness sessions  

## Screens Designed
The following mobile screens were designed:

- Home Screen  
- Programs / Services Screen  
- Login Screen  
- Book a Session Screen  
- Date and Time Slot Selection Screen  
- Booking Summary Screen  
- Booking Confirmation Screen  
- My Account / Profile Screen  
- Contact Screen  

## User Flow
Home → Explore Programs → Login → Fill Booking Form →  
Select Date and Time → View Booking Summary → Confirm Booking →  
View Session in My Account  

## Design Approach
- Mobile-first UI design  
- Clean and minimal layout  
- Clear call-to-action buttons  
- Easy navigation and readability  
- Consistent color theme and typography  

## Tools Used
- Figma for UI design and prototyping  
- Icon libraries  
- Google Fonts
